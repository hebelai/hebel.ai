# CONTRIBUTING

Danke, dass du zu Hebel.AI beitragen möchtest!

## Ablauf
1. Issue anlegen (Problem, Feature, RFC).
2. Fork/Branch, Commit mit sinnvollen Messages.
3. PR mit Beschreibung, Screenshots bei UI.
4. CI muss grün sein (Build, Lint, Tests).

## Stil
- TypeScript strict, ESLint + Prettier
- Commit-Format: Conventional Commits
- Tests: Vitest/Jest, E2E via Playwright (später)

## Lizenz
Mit dem Beitrag stimmst du der AGPL-3.0-only Lizenz zu.
