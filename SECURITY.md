# SECURITY

## Melden von Sicherheitslücken
Bitte sende vertrauliche Berichte an security@hebel.ai (Platzhalter).

## Prinzipien
- Privacy by Design: minimale Datenerhebung, Verschlüsselung im Ruhezustand und Transport.
- Pseudonymisierte Erstphase im Matching.
- Selektive Freigaben über QR/Token, zeitlich widerrufbar.
- Keys: Client-seitig generiert; Server sieht nur Public Keys.
- Backup & Rotation-Strategie (später).

## Abhängigkeiten
- Regelmäßige Dependency-Updates via Renovate.
