# Partner-Outreach – Gesprächsleitfaden (Sollit & Co.)

## Artefakte zum Teilen (GitHub-Links)
- README (Übersicht), docs/vision.md, docs/pitch/one-pager.md
- docs/legal/mou-template.md (oder finales MoU)
- AGPL-Lizenz, Governance, Security

## Gesprächsstruktur (30–45 Min)
1) Vision & Nutzen je Stakeholder (5min)
2) Sicherheits- & Open-Source-Prinzip (5min)
3) MVP & Integrationspunkte (10min)
4) Governance & Revenue-Sharing (10min)
5) Nächste Schritte: NDA → MoU → Pilot (5min)
