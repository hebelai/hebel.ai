# One-Pager – Hebel.AI

**Problem:** Fragmentierte Prozesse, Intransparenz, Datenmissbrauch (Lead-Börsen), Fachkräftemangel.

**Lösung:** Offene Plattform mit KI-Wizard → standardisierte Ausschreibung → qualifizierte Betriebe → Beraterbegleitung → verifizierte Qualität.
Datensouveränität via Keypairs/QR-Freigaben. Nachwuchs über Macher-Map.

**Partnerangebot:** Open-Source-Module (AGPL), Integration in bestehende Tools, 10% Revenue-Sharing an Busfaktor e.V.
**Pilot:** Wärmepumpe + PV, 2–3 Landkreise, 6–9 Monate.
**KPIs:** Time-to-Offer, Zuschlagsquote, Reklamationen, Jugend-Quests.
